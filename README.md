# 제작 목표
- 오픈웨더의 api를 이용하여 지역에 맞는 시간과 온도를 알아 볼 수 있는 어플을 만드는것이 목표이다.
- 오픈웨더가 배포하는 무료 api는 날씨정보만 알려주기에 한 지역의 기상예보 및 온도를 알 수 없기에 여러 나라의 날씨를 실시간으로 알수 있게 한다.
